Index is usually placed at the end of the book to help you locate the specific keywords. Index is also being used by encyclopaedia to help you locate the respective words you would like to find among the volumes. Index in a book can be replaced by several function in digital space. 

## In a book
(1) Crtl + f, find in page and find in document help you locate the keywords and let you know whether a book is relevant or not in the shortest amount of time. 

## Between books
(2) #hashtags, hashtag can be a way to organise syntopical reading and when you need to read back and forth between multiple books. There is one book that organise inn the forms of tags is call *The Syntopicon*. This is two books that organise 102  ideas that are being discussed throughout the history of western world. 

## Using Document management system or Bibliographic management software
(3) Embed a bookmark and linking. Some pdf allow you to place a bookmark into it which then allow you to copy paste them for quick references. For example, [“participatory, co-operative society.” (Fuchs, 2009, p. 243)](zotero://note/u/8KV7475U/), this then allow you to link back to the source and show on page.
